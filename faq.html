What is the purpose of this application?

The main goal of this application is to provide a safe and educational environment for users when it comes to investing.

Are there any real funds involved?

There are no real funds involved. All funds are nonexistent & virtual.

Can I buy and sell assets similar to a Brokerage/Exchange?

Users will be able to "buy" and "sell" their holdings determined by the virtual funds that the user has added and the changing monetary value of the asset.

Is there a max fund allocation amount?

~$2B USD is the max fund allocation amount. However, it is advisable to use realistic values as this application is intended to simulate a real investing environment.
